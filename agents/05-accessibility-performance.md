# 05 - Accessibility & Performance

## Accessibility
- All interactive elements must be keyboard accessible.
- Use semantic elements (button, nav, main, etc.).
- Provide `aria-*` attributes when needed.
- Ensure color contrast is acceptable in light/dark modes.

## Performance
- Use `next/image` for images.
- Keep client components minimal; use Server Components by default.
- Avoid importing large libraries unless necessary.
